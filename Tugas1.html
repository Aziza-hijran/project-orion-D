TUGAS 1

1. Apa itu version Control?
Version Control adalah sekumpulan perubahan yang terjadi karena perubahan yang kita lakukan pada satu file atau lebih dari waktu ke waktu.

2. Apa perbedaan Git dan Github?
Git : Adalah salah satu sistem pengontrol(version control).
GitHub : Adalah manajemen project, sistem versioning code sekaligus platform jaringan sosail bagi para developer seluruh dunia karena bersift online

3. Apa itu HTML?
HTML atau HyperText Markup Language adalah standar markup language yang digunakan untuk membuat dan menyusun halaman web, dengan tujuan untuk struktural pada halaman web.
